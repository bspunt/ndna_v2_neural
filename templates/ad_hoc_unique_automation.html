Unique Ad-Hoc/Free Form Automation (Run Now)

Here's how easy and powerful this is.

Just input line by line nodes and match the unique command set to each line of that node, for example, for nodes, line by line in a single column, input your nodes. The commands you would input on the "commands side" have to just be comma separated "per node/per line" e.g.

sh ip interface brie, sh ip bgp summ

You don't have to worry about spacing between commas, as the program will clean that up for you.
If I had two or more nodes, then my second line might be something like

sh ver, sh cdp nei

So, you could have something like this on your nodes form:
rt1
rt2

Then on your commands form, you'd have (Remember the system will disable pagination for you if the device is in the database!):
sh ip interface brie, sh ip bgp summ
sh ver, sh cdp nei

Tip of the day!
If you are pulling information, and your device DOESN'T exist in the neural database inventory, don't forget to include commands to disable pagination, e.g. on IOS devices using the command "term len 0" for example. If the device IS in the database inventory, neural will take care of disabling pagination for you!
