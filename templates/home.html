Services Provided by NEURAL

Using Neural, you'll be able to do all kinds of network automation.

In addition, Neural is a full featured configuration templating system.


See below for more details on what Neural will provide!

All Kinds Of Automation!

Including: free-form, Unique and self-service (Pre-Built-Programs using SSH or REST APIs). No coding skills needed! You can choose to "run-now" or "schedule later" for any type of automation.


Automation That's Fast!

Perform Analysis or Programming at Scale, with no limit on output file size, threading, plus writing the output to separate files for each node and writing job results to MySQL



Config Templating using Jinja2/CSV files

No coding skills needed! Couldn't be easier! Just upload your CSV w/ variables filled in, then choose the device platform, and create your templates. You can then push out to the network or download your configs (single file or all via zip file).

Golden Configurations For Compliance

No coding skills needed! Couldn't be easier!
Just upload your Golden Configurations via an easy to use webform. .

Run Compliance Checks At Scale!

You can run compliance checks against the production network on a single device or at scale!

Config Backups and Config Diffs

You can set weekly backups and then run compare diffs on configs to see changes!
